== Description ==

LINE 聊天機器人外掛, 透過 LINE Messaging API發送訊息

###此外掛為WooCommerce擴充外掛，請先安裝WooCommerce
###必須配合Line登入外掛，請先安裝Line登入外掛，目前支持
1.Super Socializer
2.Nextend Social Login and Register
3.HB Line Login Tiny For Woocommerce

###主要功能
*新訂單事件
*狀態事件
*訂單查詢事件

###新訂單事件
當有新訂單時發送訊息

###狀態事件
當狀態改變時，如“完成”發送訊息

###訂單查詢事件
支援客戶輸入指定文字(如訂單查詢)進行訂單查詢(不須耗Messaging API次數)
